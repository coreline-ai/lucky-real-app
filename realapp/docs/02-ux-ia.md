# UX/IA 명세

## UX 원칙

오행가디언즈는 만세력 엔진의 해석을 “오늘 내가 바로 이해하고 실천할 수 있는 수호 루틴”으로 바꾼다.

핵심 경험:

- 오늘의 수호신 확인
- 오늘 운세 읽기
- 오행 루틴 수행
- 카드/도감 수집
- 시장관찰 체크리스트
- 궁합/케미 공유

금지 경험:

- 광고 보기 보상
- 구독 가입
- 유료 리포트 구매
- 유료 카드팩 구매
- 결제 복원
- 프리미엄 회원 전용 탭
- 광고 제거 상품
- 기간 한정 할인
- 가격 앵커링
- 과금 유도형 희소성 문구

## 정보 구조

하단 탭은 6개를 기본으로 한다.

| 탭 | 역할 | 대표 CTA |
|---|---|---|
| 홈 | 오늘의 수호신, 오늘 운세, 핵심 CTA | 루틴 시작 |
| 운세 | 일간/주간 흐름, 오행별 해석 | 자세히 보기 |
| 루틴 | 오늘의 오행 루틴, 수행 기록 | 완료 체크 |
| 도감 | 수호신 카드, 보상, 무료 테마/스킨 | 카드 확인 |
| 시장관찰 | 관심종목 검색, 오늘 리듬, 체크리스트 기반 관찰 후보 | 관심종목 추가 |
| 케미 | 궁합/케미 보기, 공유 이미지 생성 | 공유 카드 만들기 |

설정, 프로필, 알림 관리, 데이터 수정은 홈 상단 또는 마이 영역에서 진입한다. 결제, 구독, 광고 제거, 프리미엄 탭은 만들지 않는다.

## 온보딩 플로우

### 1. 앱 소개

3장 이내 슬라이드:

1. 오늘의 수호신을 만난다.
2. 나에게 맞는 오행 루틴을 시작한다.
3. 기록과 도감으로 하루를 쌓는다.

만세력, 오행, 십신, 루틴, 도감 보상을 한 화면에서 모두 설명하지 않는다.

### 2. 출생 정보 입력

입력 필드:

- 닉네임
- 생년월일
- 출생 시간 또는 모름
- 양력/음력
- 음력 윤달 여부
- 성별 또는 전통 계산용 성향값 (미선택 시 대운 해석 제외 안내)
- 시간대: Asia/Seoul 고정 (선택 기능은 v2)

출생 시간을 모르는 경우:

- 시주 계산이 제한될 수 있음을 부드럽게 안내한다.
- 앱 사용을 막지 않는다.
- 시간 기반 해석에는 “출생 시간 미상 기준” 라벨을 표시한다.

### 3. 첫 수호신 공개

첫 보상 경험:

- 수호신 카드 공개 연출
- 짧은 해석
- 오늘의 추천 루틴 CTA
- 도감에 첫 카드 저장

### 4. 알림 권한 요청

권한 요청은 앱 실행 직후가 아니라 첫 수호신 공개 후 노출한다.

옵션:

- 매일 아침 수호신 알림 받기
- 저녁 루틴 리마인드 받기
- 나중에 설정하기

권한 거부 시:

- 앱 사용을 막지 않는다.
- 홈 또는 설정에서 다시 켤 수 있게 안내한다.

## 홈 화면

홈은 데일리 허브다.

구성:

- 오늘 날짜
- 오늘의 수호신 카드
- 한 줄 운세
- 오행 밸런스 요약
- 오늘 추천 루틴
- 카드 보상 상태
- 케미 공유 진입점

상태별 UI:

| 상태 | UI |
|---|---|
| 첫 방문 | 수호신 공개 CTA와 입력 완료 안내 |
| 일반 | 오늘의 수호신, 운세, 루틴 CTA |
| 루틴 완료 | 완료 배지, 보상 카드 CTA |
| 엔진 계산 실패 | 재시도 버튼과 기본 안내 |
| 출생 정보 부족 | 정보 보완 CTA |

홈 화면의 가장 중요한 CTA는 **루틴 시작**이다. 사용자가 운세를 읽고 바로 행동으로 넘어가도록 만든다.

날짜 경계 규칙:

- 앱 사용 중 자정(Asia/Seoul 기준)이 지나면 다음 화면 진입 또는 새로고침 시 오늘 데이터를 새 날짜 기준으로 갱신한다.
- 갱신 전에 완료한 루틴/보상 기록은 이전 날짜에 귀속된다.

## 운세 화면

운세 화면은 오늘 운세를 더 깊게 읽는 공간이다. 만세력 엔진 결과를 그대로 노출하지 않고 사용자가 이해하기 쉬운 문장 구조로 변환한다.

권장 구성:

- 총운
- 감정 흐름
- 관계 흐름
- 일/학업 흐름
- 건강/컨디션 흐름
- 오늘 조심할 점
- 오늘의 오행 키워드
- 전문 용어 접힌 설명

문구 원칙:

- 단정 대신 경향으로 표현한다.
- 낮은 점수도 “나쁜 날”이라고 쓰지 않는다.
- 회복, 정리, 조절, 선택 같은 언어를 사용한다.
- 건강/투자/법률 판단으로 보일 수 있는 문구를 피한다.

리포트:

- “프리미엄 리포트”가 아니라 “주간 회고 리포트”, “이번 달 오행 리포트”, “수호신 성장 기록”처럼 무료 콘텐츠로 정의한다.

## 루틴 화면

루틴 화면은 오행별 추천 행동을 실천하는 공간이다.

루틴 예시:

| 오행 | 루틴 방향 | 예시 |
|---|---|---|
| 목 | 성장, 계획 | 오늘 할 일 3개 적기, 10분 산책 |
| 화 | 표현, 활력 | 감사 메시지 보내기, 가벼운 스트레칭 |
| 토 | 안정, 정리 | 책상 정리 5분, 식사 천천히 하기 |
| 금 | 판단, 집중 | 알림 정리, 미룬 결정 하나 처리 |
| 수 | 회복, 흐름 | 물 마시기, 조용히 호흡하기 |

상태별 UI:

| 상태 | UI |
|---|---|
| 미시작 | 오늘 추천 루틴과 시작 버튼 |
| 진행 중 | 체크리스트, 남은 항목 |
| 완료 | 완료 연출, 카드 보상 안내 |
| 건너뛰기 | 가벼운 확인 후 홈 복귀 |
| 연속 수행 | 스트릭, 무료 테마/카드 해금 안내 |

미완료 페널티는 없다. 루틴은 사용자를 압박하지 않는다.

## 도감 화면

도감은 수호신 카드와 무료 보상을 모으는 공간이다.

카드 획득 방식:

- 오늘 접속
- 루틴 완료
- 연속 기록
- 시즌 이벤트
- 케미 공유
- 특정 오행 균형 달성

카드팩 정의:

- “오늘의 수호 팩”
- “오행 균형 팩”
- “시즌 기운 팩”

위 카드팩은 모두 무료 해금 콘텐츠다. 구매 버튼, 가격표, 유료 잠금 UI, 희소성 압박 문구는 사용하지 않는다.

## 케미 화면

케미 화면은 나와 친구, 연인, 가족, 동료의 오행 궁합을 가볍게 확인하고 공유하는 공간이다.

입력 방식:

- 상대 생년월일시 직접 입력
- 저장된 임시 프로필 선택
- 출생 시간 모름 허용

결과 구성:

- 케미 점수
- 잘 맞는 기운
- 서로 보완되는 지점
- 조심하면 좋은 대화 방식
- 함께하면 좋은 루틴
- 공유 카드 미리보기

공유용 이미지 기본값:

- 내 닉네임 (기본 표시, 공유 전 숨김 선택 가능)
- 케미 타입
- 대표 수호신
- 짧은 해석

공유 이미지에 기본 포함하지 않는 정보:

- 생년월일 전체
- 출생 시간
- 개인 메모
- 상세 사주팔자
- 상대방 닉네임/이름 (사용자가 명시적으로 켤 때만 포함)

## 공유 플로우

공유는 앱 확산보다 사용자의 자기표현을 우선한다.

공유 가능한 콘텐츠:

- 오늘의 수호신 카드
- 오늘 운세 한 줄
- 루틴 완료 카드
- 케미 결과 카드
- 도감 획득 카드

공유 단계:

1. 공유 대상 선택
2. 공유 미리보기 생성
3. 닉네임 표시 여부 선택
4. 민감 정보 숨김 확인
5. OS 공유 시트 호출
6. 취소 또는 완료 후 앱 복귀

공유 후 보상은 무료 수집 요소로만 제공할 수 있다. 광고 보기나 결제와 연결하지 않는다.

## 알림 플로우

알림은 사용자의 하루 흐름을 돕는 용도로만 사용한다.

MVP 알림 종류:

- 매일 아침 오늘의 수호신 알림
- 저녁 루틴 리마인드
- 카드 보상 미수령 알림

MVP 이후 확장 알림 (01 무료 확장 기능과 연동):

- 점심 오행 밸런스 체크
- 주간 회고 알림

설정:

- 알림 전체 끄기
- 알림 종류별 켜기/끄기
- 시간 변경
- 방해금지 시간대 설정

서버 푸시 없이 로컬 예약 알림을 기본으로 한다.

## 설정 및 프로필

설정 화면 항목:

- 출생 정보 수정
- 닉네임 변경
- 알림 관리
- 해석 톤 변경
- 테마 변경
- 데이터 초기화
- 개인정보 안내
- 엔터테인먼트 목적 안내
- 앱 버전/엔진 버전

테마:

- 기본 테마
- 오행 테마
- 시즌 테마
- 도감 해금 테마

테마는 무료 설정 항목 또는 도감 보상이다. 결제 잠금 상태를 만들지 않는다.

## 빈 상태와 오류 상태

빈 상태:

- 도감 없음: “오늘의 루틴을 완료하고 첫 수호신 카드를 받아보세요.”
- 케미 상대 없음: “친구의 생년월일을 입력해 첫 케미를 확인하세요.”
- 기록 없음: “오늘의 수호신을 확인하면 첫 기록이 쌓여요.”

오류 상태:

- 엔진 계산 실패: 재시도와 입력 정보 확인 CTA
- 지원 범위 밖 날짜: 지원 범위 안내
- 잘못된 음력/윤달 입력: 입력 수정 안내
- 저장 실패: 다시 저장하기, 임시 결과 유지
- 공유 이미지 생성 실패: 재시도 안내, 텍스트 공유 대안 제시

오류 화면에는 스택트레이스나 내부 오류 코드를 그대로 노출하지 않는다.
