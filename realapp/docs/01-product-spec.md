# 제품 명세

## 제품 비전

**오행가디언즈**는 만세력 엔진을 어렵고 무겁게 소비하는 대신, 매일의 기운을 쉽고 다정한 스마트폰 경험으로 바꾸는 무료 앱이다.

앱은 사용자가 오늘의 수호신, 오늘 운세, 오행 루틴, 카드/도감 보상, 궁합/케미 공유를 통해 자신의 하루를 가볍게 해석하고 실천으로 연결하도록 돕는다.

목표는 “정답을 알려주는 점술 앱”이 아니다. 목표는 사용자가 오늘의 흐름을 이해하고 스스로를 돌보는 **일상형 오행 가이드**가 되는 것이다.

## 포지셔닝

오행가디언즈는 전통 만세력 기반 정보를 현대적인 모바일 루틴과 캐릭터 보상 경험으로 재해석한다.

포지셔닝 문장:

> 오행가디언즈는 만세력 기반 오늘의 흐름을 캐릭터 수호신, 오행 루틴, 카드 도감, 궁합 공유로 풀어내는 무료 일상 운세 앱이다.

기존 운세 앱과 다른 점:

- 결과 소비보다 하루 루틴 실천을 강조한다.
- 결제 유도보다 무료 신뢰를 우선한다.
- 공포스럽거나 단정적인 예언보다 자기이해와 회고를 제공한다.
- 만세력 표를 첫 화면에 내세우지 않고, 오늘 할 수 있는 행동을 먼저 보여준다.
- 카드/도감은 결제 상품이 아니라 재방문과 기록의 보상이다.

## 핵심 사용자

주요 사용자군:

- 매일 아침 오늘 운세와 컨디션 힌트를 확인하고 싶은 사용자
- 사주, 오행, 만세력에 입문하고 싶은 사용자
- 전문 용어보다 쉬운 해석과 실천 팁을 원하는 사용자
- 캐릭터, 카드, 도감 수집형 보상에 동기부여를 느끼는 사용자
- 친구, 연인, 가족과 궁합이나 케미를 재미있게 공유하고 싶은 사용자
- 무겁거나 불안감을 조성하는 운세 앱보다 밝고 건강한 해석을 원하는 사용자

보조 사용자군:

- 일기/회고 앱을 쓰지만 꾸준히 기록하기 어려운 사용자
- 루틴 앱을 쓰고 싶지만 딱딱한 체크리스트가 부담스러운 사용자
- SNS나 메신저로 가벼운 운세 카드를 공유하는 사용자

## 핵심 가치

- **쉽게 이해되는 만세력**: 전문 용어를 최소화하고 오늘의 기운을 일상 언어로 설명한다.
- **매일 돌아오는 이유**: 오늘의 수호신, 오늘 운세, 루틴, 보상으로 자연스러운 재방문 동기를 만든다.
- **해석에서 실천으로**: 운세를 단순히 읽는 데서 끝내지 않고 오행 루틴으로 연결한다.
- **수집의 즐거움**: 카드와 도감 보상을 통해 사용자가 자신의 흐름과 기록을 쌓는 재미를 느낀다.
- **가벼운 관계 공유**: 궁합과 케미를 부담 없는 대화 소재로 공유할 수 있게 한다.
- **불안 조장이 아닌 자기 돌봄**: 부정적 예언이나 결핍 유도 대신 균형, 회복, 선택의 관점을 제공한다.

## 핵심 데일리 루프

1. 아침 알림 또는 앱 실행으로 진입한다.
2. 홈에서 오늘의 수호신과 한 줄 운세를 확인한다.
3. 오행 밸런스와 분야별 운세를 짧게 본다.
4. 오늘의 추천 루틴 1-3개 중 하나를 선택한다.
5. 루틴 완료 여부와 감정 체크를 기록한다.
6. 루틴 완료 또는 일일 확인으로 무료 카드/도감 보상을 받는다.
7. 원하면 친구/연인과 오늘의 케미 카드를 공유한다.
8. 기록 캘린더에 오늘의 수호신, 루틴, 감정, 보상이 쌓인다.

핵심은 “운세를 봤다”에서 끝나는 것이 아니라 **오늘 할 행동 하나를 고르고, 체크하고, 보상을 받는 구조**다.

## MVP 기능 범위

### 온보딩/프로필

- 앱 소개 3장 이내
- 닉네임 입력
- 생년월일 입력
- 출생 시간 입력 또는 모름 선택
- 양력/음력 선택
- 음력 윤달 여부 선택
- 성별 또는 전통 계산용 성향값 선택
- 알림 권한은 첫 수호신 공개 후 요청

### 오늘 홈

- 오늘 날짜
- 오늘의 수호신 카드
- 한 줄 운세
- 오행 밸런스 요약
- 총운/관계운/행동 조언
- 오늘 추천 루틴 1-3개
- 카드 보상 상태
- 케미 공유 진입점

### 운세

- 오늘 총운
- 감정 흐름
- 관계/소통 흐름
- 일/학업 흐름
- 컨디션 흐름
- 오늘 조심할 점
- 오늘의 오행 키워드
- 전문 용어 접힌 설명

### 루틴

- 목/화/토/금/수 기반 루틴 템플릿
- 오늘 추천 루틴 체크
- 완료 기록 저장
- 7일 흐름 요약
- 미완료 페널티 없음

### 카드/도감

- 수호신 카드
- 오행 카드
- 카드 상세 화면
- 도감 진행률
- 일일 확인 보상
- 루틴 완료 보상
- 중복 카드 처리

### 궁합/케미

- 상대 프로필 입력
- 관계 타입 선택: 친구, 연인, 가족, 동료, 기타
- 오행/십신 기반 케미 요약
- 잘 맞는 지점
- 조심할 소통 방식
- 함께 하기 좋은 루틴
- 공유 이미지 생성
- 공유 전 민감정보 숨김 기본값

### 기록/설정

- 기록 캘린더
- 감정/컨디션 기록
- 출생정보 수정
- 알림 시간 설정
- 무료 테마 설정
- 데이터 삭제
- 개인정보 안내

## 무료 기능 원칙

오행가디언즈는 수익화를 하지 않는다. 사용자의 불안, 궁금증, 관계 고민을 결제 전환 수단으로 사용하지 않는다.

운영 원칙:

- 핵심 운세와 루틴 경험은 항상 무료로 제공한다.
- 해석을 일부러 숨기거나 과장하지 않는다.
- 부정적 결과를 공포스럽게 표현하지 않는다.
- 개인정보와 생년월일시는 기능 제공에 필요한 범위에서만 다룬다.
- 공유 기능은 사용자의 자발적 선택을 전제로 한다.
- 카드, 도감, 테마, 리포트는 보상과 콘텐츠 확장으로 설계한다.
- 운영 부담을 줄이기 위해 콘텐츠 구조는 재사용 가능하고 확장 가능한 형태로 관리한다.

## 무료 확장 기능

MVP 이후 확장 후보:

- 주간 회고 리포트
- 월간 오행 리포트
- 시즌별 무료 카드팩
- 7일 오행 루틴 챌린지
- 30일 도감 완성 이벤트
- 친구/커플 케미 히스토리
- 가족/팀 케미 모드
- 무료 테마 보상
- 수호신별 배경 이미지
- 용어 사전
- 초보자 해석 모드
- 홈 위젯
- 다국어 지원

## 제품 판단 기준

기능을 추가할 때 다음 질문을 통과해야 한다.

- 사용자의 하루를 더 잘 이해하게 돕는가?
- 불안이 아니라 자기 돌봄으로 이어지는가?
- 무료 앱의 신뢰를 해치지 않는가?
- 반복 사용, 수집, 공유 중 하나 이상의 즐거움을 만드는가?
- 만세력 기반 앱이라는 정체성을 강화하는가?
- 개인정보 노출을 늘리지 않는가?
- 결제/광고/구독 구조 없이도 유지 가능한가?
