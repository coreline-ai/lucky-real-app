## 개발 목적

오행가디언즈 Flutter 앱의 1차 개발 범위인 **1인 사용자 데일리 코어 루프**를 구현한다.

> 온보딩 → 오늘의 수호신/운세 확인 → 루틴 체크 → 일일 카드 보상 → 기록 저장

기존 TypeScript 만세력 엔진을 기준 구현으로 삼아 Dart 엔진을 포팅하고, fixture 전건 일치를 정확도 게이트로 사용한다.
