# implement_20260708_141545.md

작성 일시: `2026-07-08 14:15:45 KST`

이 문서는 이번 개발의 범위를 고정하고, 구현이 목적 밖으로 확장되지 않도록 하기 위한 작업 문서다.

## 개발 목적

이전 전문가 리뷰 항목을 현재 HEAD 기준으로 재검증하고, 이미 해결된 결함은 회귀 보호 대상으로 고정한다. 남은 작업은 시장관찰 미구현 범위, 에셋/문서 정책 정리, 최종 QA 게이트로 제한한다.

## 개발 범위

- 이전 리뷰 항목 7건을 현재 코드와 테스트 기준으로 재분류한다.
- 정적 분석/테스트 게이트 결과를 계획 문서에 기록한다.
- 데이터 삭제, 공유 취소, 루틴 날짜 전환, v3 마이그레이션은 재구현하지 않고 회귀 보호 기준만 정한다.
- 시장관찰/종목 데이터 기능은 기존 계획 문서와 실제 구현물의 차이를 추적한다.
- WebP 런타임 에셋, PNG 원본, imagegen source의 보관/커밋 정책을 정리한다.
- 구현 상태와 어긋난 주석/문서를 실제 기능 상태에 맞게 정리한다.

## 제외 범위

- 새로운 투자/추천 알고리즘 추가
