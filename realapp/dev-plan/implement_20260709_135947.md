## 개발 목적

홈 탭의 오늘 수호신 카드 안에서 공유 CTA가 눈에 잘 띄지 않는 문제를 개선한다.

수호신 공유 버튼 위에 `오늘의 행운 숫자` 3개를 추가하고, 숫자 영역과 공유 버튼이 홈 카드 분위기에 자연스럽게 섞이도록 이미지젠 기반 에셋을 만든다. 숫자는 이미지에 박지 않고 Flutter 텍스트로 렌더링해 사용자/날짜/만세력 결과에 따라 결정적으로 달라지게 한다.

## 개발 범위

- 홈 수호신 카드 내부에 오늘의 행운 숫자 3개를 표시한다.
- 행운 숫자는 만세력 결과에서 나온 날짜, 일진, 수호신 오행, 십신, 사용자 프로필을 seed로 사용해 결정적으로 생성한다.
- 행운 숫자는 오락·자기성찰용 리듬 숫자로만 표현하고, 로또/투자/도박/수익 예측처럼 보이지 않게 한다.
- 기존 작은 `공유` 텍스트 버튼을 `오늘 수호신 공유하기` CTA로 더 눈에 띄게 바꾼다.
- 이미지젠 스킬로 홈 카드 안에 들어갈 배경 에셋 2종을 생성한다.
  - `home_lucky_numbers_plate_960x260_v1`
  - `home_share_cta_glow_960x180_v1`
- 생성 원본은 `assets/imagegen_sources/masters`, 런타임 WebP는 `assets/images/home`에 저장한다.
- `AssetPaths`, `pubspec.yaml`, asset registration test를 업데이트한다.
- 홈 스모크 테스트와 행운 숫자 helper 테스트를 추가한다.

## 확인 필요 사항 / 결정 기록

- 숫자 범위는 `1~99`로 한다.
- 숫자는 중복 없는 3개로 정렬 표시한다.
- seed는 `profile.id`, `date`, `daily.dayPillar.ganji`, `match.element`, `match.todayElement`, `analysis.dayStemSipsin`을 사용한다.
- 숫자 생성은 순수 Dart helper로 분리해 테스트한다.
- 이미지에는 숫자나 한글 문구를 넣지 않는다. 텍스트는 Flutter에서 렌더링한다.
