## 개발 목적

생성 완료된 P0 에셋 72종(전건 존재 확인)을 1차 앱에 연결한다. v1 임시 에셋을 v2로 교체하고, 화면별 배경·카드 아트·빈/오류 상태 일러스트를 적용해 앱의 시각 완성도를 1차 출시 수준으로 올린다.

## 개발 범위

- pubspec 에셋 등록 확장: `effects/`, `collection/`, `errors/`, `history/`, `settings/` 폴더 추가
- `asset_paths.dart` v2 전환: 수호신 양 5종(1024 v1 → 2048 v2), 카드 프레임/뒷면(1024x1536 → 1080x1620 v2)
- 신규 경로 상수 추가: 오행 카드·수호신 카드 아트, 운세/루틴/홈 패널 배경, element card/wide 변형, 완료 뱃지, 빈/오류 일러스트
- 화면 적용 (기능 변경 없이 비주얼만):
  - 도감: 카드 타일·상세에 전용 카드 아트(`card_element_*`, `card_guardian_*_yang`), 미보유는 v2 카드 뒷면
  - 홈: 수호신 영역에 오행 card 배경, 밸런스 패널 배경
  - 운세 탭: 상단 배너(`fortune_total_bg`)
  - 루틴 탭: 수호신 오행별 상단 배너(`routine_{element}_bg`), 전체 완료 시 뱃지(`routine_complete_badge`)
  - 빈/오류 상태: 기록 빈 상태, 엔진 계산 실패 일러스트
- 런처 아이콘 적용: BRAND-001 → Android mipmap (`flutter_launcher_icons`)
- 에셋 로딩 테스트를 신규 경로 전체로 확장
