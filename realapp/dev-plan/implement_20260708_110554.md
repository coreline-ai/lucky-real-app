## 개발 목적

탭별 배경 이미지 적용(B안·완성형): 단색 Scaffold로 밋밋한 화면에 프로젝트 정체성에 맞는 배경을 깔되, **홈은 오늘의 수호신 오행에 따라 매일 바뀌는 동적 배경**으로 "매일 다른 하루"를 시각화한다. 텍스트 가독성은 공통 스크림 구조로 보장한다.

## 개발 범위

- 공통 `TabBackground` 위젯 1개: 배경 이미지 + 상→하 스크림 그라데이션(가독성 보장) + `cacheWidth` 디코드 최적화
- 탭/화면별 적용 (아래 "필요 에셋 정의" 표 기준):
  - 홈: 오늘 수호신 오행별 세로 배경 5종 (동적), 프로필 없음 상태는 `home_bg_default`
  - 루틴: 홈과 동일 오행 배경(저투명 스크림) — 하루의 연속성
  - 도감: `element_bg_wide` 저투명 (그리드가 주인공, 절제 적용)
  - 운세: 상단 배너 유지 + 관계/일학업/컨디션 섹션 카드에 전용 배경 스트립 3종
  - 기록: `history_calendar_bg`, 설정: `settings_my_page_bg`
- WebP 파생 5종 생성(에셋 도구 `optimize_apk_assets_webp.cjs`) 및 pubspec 파일 단위 등록
- 카드/텍스트 컨테이너 반투명 surface 조정 (기존 위젯 구조 유지, 스타일만)
- 에셋 로딩 테스트 확장 + 가독성·오버플로 회귀 테스트
